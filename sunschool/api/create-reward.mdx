---
title: "Create reward"
api: "POST /api/rewards"
description: "Create a new reward in the family shop"
---

<Note>Requires **PARENT** or **ADMIN** role.</Note>

## Request

```json
{
  "title": "Ice Cream Trip",
  "description": "A trip to the ice cream shop",
  "tokenCost": 50,
  "category": "treats",
  "emoji": "🍦",
  "color": "#FF8C42",
  "maxRedemptions": 5
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `title` | string | Yes | Reward name |
| `description` | string | No | Reward description |
| `tokenCost` | number | Yes | Point cost to redeem |
| `category` | string | No | Category for grouping |
| `emoji` | string | No | Display emoji |
| `color` | string | No | Display color (hex) |
| `maxRedemptions` | number | No | Maximum total redemptions (null = unlimited) |

## Response

Returns the created reward object.
