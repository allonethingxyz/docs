---
title: "Delete learner"
api: "DELETE /api/learners/{id}"
description: "Delete a child learner account"
---

<Note>Requires **PARENT** or **ADMIN** role.</Note>

Deletes a learner account and all associated data (lessons, quiz answers, achievements, points).

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | number | Learner user ID |

## Response

```json
{
  "message": "Learner deleted successfully"
}
```

### Errors

| Status | Description |
|--------|-------------|
| 403 | Not authorized (learner doesn't belong to this parent) |
| 404 | Learner not found |
