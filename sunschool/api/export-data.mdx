---
title: "Export data"
api: "GET /api/export"
description: "Export all family learning data"
---

<Note>Requires **PARENT** or **ADMIN** role.</Note>

Exports all data for the authenticated parent's family: learner profiles, lessons, quiz answers, achievements, and points.

## Response

Returns a JSON object containing all exportable data for the parent's account and all linked learner accounts.

```json
{
  "parent": { ... },
  "learners": [
    {
      "profile": { ... },
      "lessons": [...],
      "achievements": [...],
      "points": { ... }
    }
  ],
  "exportedAt": "2026-03-11T00:00:00.000Z"
}
```
