---
title: "Get lesson"
api: "GET /api/lessons/{lessonId}"
description: "Get a specific lesson by ID"
---

<Note>Requires authentication.</Note>

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `lessonId` | string | Lesson UUID |

```json
  "learnerId": 5,

  "difficulty": "beginner",
  "score": null,
  "spec": {
    "title": "Understanding Fractions",
    "sections": [...],
    "questions": [...],
    "images": [...],
    "diagrams": [...]
  },
  "createdAt": "2026-03-11T00:00:00.000Z",

```

### Errors

| Status | Description |
|--------|-------------|
| 404 | Lesson not found |
