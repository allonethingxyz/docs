---
title: "Get reports"
api: "GET /api/reports"
description: "Get analytics and progress reports"
---

<Note>Requires authentication.</Note>

Returns learning analytics for the specified learner, including lesson history, quiz performance, and concept mastery.

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `learnerId` | number | Learner ID to generate report for |

## Response

Returns aggregated analytics data including lessons completed, average scores, and performance trends.
