---
title: "List lessons"
api: "GET /api/lessons"
description: "List all lessons for a learner"
---

<Note>Requires authentication.</Note>

## Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `learnerId` | number | Filter by learner ID |
| `status` | string | Filter by status: `QUEUED`, `ACTIVE`, `DONE` |

## Response

```json
[
  {
    "id": "d0343503-f02a-4335-bcf5-eec6ed871942",
    "learnerId": 5,
    "status": "DONE",
    "subject": "Math",
    "category": "Fractions",
    "score": 100,
    "createdAt": "2026-03-11T00:00:00.000Z",
    "completedAt": "2026-03-11T00:05:00.000Z"
  }
]
```
