---
title: "List rewards"
api: "GET /api/rewards"
description: "List available rewards"
---

<Note>Requires authentication.</Note>

Returns available rewards. Parents see their own created rewards. Learners see active rewards available for redemption.

## Response

```json
[
  {
    "id": 1,
    "parentId": 1,
    "title": "Extra Screen Time",
    "description": "30 minutes of extra screen time",
    "tokenCost": 20,
    "category": "privileges",
    "isActive": true,
    "maxRedemptions": null,
    "currentRedemptions": 2,
    "emoji": "📺",
    "color": "#4A90D9"
  }
]
```
