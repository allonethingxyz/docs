---
title: "Register"
api: "POST /register"
description: "Create a new user account"
---

## Request

```json
{
  "username": "parent1",
  "email": "parent@example.com",
  "name": "Jane Doe",
  "role": "PARENT",
  "password": "securepassword"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `username` | string | Yes | Unique username |
| `email` | string | Yes | Email address |
| `name` | string | Yes | Display name |
| `role` | string | Yes | `ADMIN`, `PARENT`, or `LEARNER` |
| `password` | string | Yes | Account password |
| `parentId` | number | No | Parent user ID (for learner accounts) |

<Note>The first user to register on a new instance is automatically promoted to Admin regardless of the requested role.</Note>

## Response

```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": 1,
    "username": "parent1",
    "email": "parent@example.com",
    "name": "Jane Doe",
    "role": "PARENT"
  }
}
```

### Errors

| Status | Description |
|--------|-------------|
| 400 | Missing required fields |
| 400 | Invalid role |
| 400 | Username already exists |
