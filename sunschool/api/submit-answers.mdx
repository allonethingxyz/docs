---
title: "Submit answers"
api: "POST /api/lessons/{lessonId}/answer"
description: "Submit quiz answers and receive scores"
---

<Note>Requires authentication.</Note>

Submits the learner's quiz answers, calculates scores, awards points, and checks for achievements. The lesson status changes to `DONE` after submission.

## Path parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `lessonId` | string | Lesson UUID |

## Request

```json
{
  "learnerId": 5,
  "answers": [0, 2, 1],
  "wagers": [0, 5, 0]
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `learnerId` | number | Yes | Learner ID |
| `answers` | number[] | Yes | Array of selected option indices (one per question) |
| `wagers` | number[] | No | Array of point wagers for double-or-nothing (0 = no wager) |

## Response

```json
{
  "score": 67,
  "correct": 2,
  "total": 3,
  "pointsEarned": 7,
  "results": [
    { "correct": true, "pointsEarned": 1 },
    { "correct": false, "pointsEarned": -5 },
    { "correct": true, "pointsEarned": 1 }
  ],
  "achievements": [
    {
      "type": "FIRST_LESSON",
      "payload": { "lessonId": "d034..." },
      "tokenReward": 5
    }
  ]
}
```

### Scoring

- **Correct answer (no wager):** +1 point
- **Correct answer (with wager):** +wager amount
- **Incorrect answer (with wager):** -wager amount
- **Achievement bonus:** Additional points per achievement unlocked
