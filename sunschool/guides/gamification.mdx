## Points system

Children earn points through learning activities:

| Source | Points |
|--------|--------|
| Correct quiz answer | 1 point per correct answer |
| Double-or-nothing (correct) | Double the wagered amount |
| Double-or-nothing (incorrect) | Lose the wagered amount |
| Achievement unlock | Varies by achievement type |

Points are tracked in a ledger with full history. The current balance is always available at `GET /api/points/balance`.

## Rewards

Parents create and manage a rewards shop. Each reward has:

- **Title and description** — What the reward is
- **Point cost** — How many points it costs to redeem
- **Category** — For organizing rewards
- **Max redemptions** — Optional limit on how many times it can be redeemed

### Redemption workflow

1. Child browses available rewards at their **Goals** page
2. Child requests a redemption (points are held pending)
3. Parent sees the request in **Rewards Management**
4. Parent approves or rejects with optional notes
5. On approval, points are deducted; on rejection, points are returned

## Achievements

Achievements are awarded automatically for milestones:

- **First Lesson** — Complete your first lesson
- **Perfect Score** — Get 100% on a quiz
- **Streak** — Complete lessons on consecutive days
- **Subject milestones** — Complete N lessons in a subject

Each achievement can include a token reward (bonus points). Achievements can be shared via a public URL (`/users/:username/award/:hash`).

## Goal saving

Children can save toward specific rewards:

1. Pick a reward from the available list
2. Allocate points toward it over time
3. Once the goal is met, request redemption

This teaches delayed gratification and budgeting alongside the academic content.

## Double-or-nothing

An optional wager system on quiz questions:

- Before answering, the child can choose to wager points
- Correct answer: double the wagered points
- Incorrect answer: lose the wagered points
- Parents can enable/disable this per child via `PUT /api/learner-settings/:learnerId/double-or-loss`
