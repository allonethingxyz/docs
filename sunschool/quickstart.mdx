## Use the hosted version

The fastest way to get started:

<Steps>
  <Step title="Sign up">
    Go to [sunschool.xyz](https://sunschool.xyz) and create an account with your email. No credit card required.
  </Step>
  <Step title="Add your child">
    Enter their name and grade level. The AI takes it from there.
  </Step>
  <Step title="Start learning">
    Pick a subject and generate the first lesson. Your child can start immediately.
  </Step>
</Steps>

<Tip>The first user to register on a new instance is automatically promoted to Admin.</Tip>

## User roles

| Role | Access |
|------|--------|
| **Admin** | Full system access, user management, all data |
| **Parent** | Manage children, view progress, configure rewards |
| **Learner** | Access lessons, take quizzes, view achievements, redeem rewards |

Parents create learner accounts for their children. Each child gets their own adaptive learning path.

## What happens next

Once a child has an account:

1. **Pick a subject** — Math, Science, History, English, or any custom topic
2. **AI generates a lesson** — Grade-appropriate content with illustrations, tailored to the child
3. **Take the quiz** — 3 multiple-choice questions with optional double-or-nothing wagering
4. **Earn points** — Correct answers earn points that can be spent on parent-managed rewards
5. **Track progress** — Parents see scores, achievements, and concept mastery in real time
